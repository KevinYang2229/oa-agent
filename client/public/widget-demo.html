這是一個「別人的網站」

這個頁面模擬第三方網站。它本身與 OA 系統無關，只在頁尾加入了一行 widget 載入碼，右下角就會出現浮動按鈕；點擊後彈出聊天視窗即可串接後端 AI。

最簡接入（對方只需貼這一行，落到預設租戶，向後相容）：

<script src="https://你的網域/widget.js"></script>

產品化接入（多租戶、預選表單、語言、外觀、SSO）：

<script src="https://你的網域/widget.js" data-title="客服小幫手" data-key="pk_xxxxx" data-form="leave-request" data-locale="zh-Hant" data-theme="light" data-position="right" data-user-token="<宿主簽發的 JWT>"></script>

宿主頁可監聽事件、或程式化控制：

// 監聽「表單已送出」 window.addEventListener('oa-agent:submitted', function (e) { console.log('已送出', e.detail.submission); }); // 程式化開關 OAAgent.open(); OAAgent.close(); OAAgent.toggle();
