試試畫面 右下角 的小幫手 ↘

⚡ 一行嵌入

<script src=".../widget.js"> 貼上即用，純 JS、零依賴、不必打包。

💬 浮動按鈕

右下角小幫手，點擊彈出聊天室；不干擾原頁面排版。

🔐 SSO 免登入

帶 data-user-token 沿用宿主登入身分，使用者免再登入。

🎨 可設定外觀

data-title / data-theme / data-locale / data-position 調整文案、亮暗、語言、左右。

🛡️ 樣式隔離

聊天介面跑在 iframe 內，scoped 樣式不污染宿主頁，不會被你的 CSS 影響。

🧩 事件可監聽

送出等事件以 oa-agent:submitted 派發，宿主頁可接手後續流程。

怎麼接？三步

  1. 跟我們申請租戶，拿到公開金鑰 pk_…。
  2. （選用）你的後端用租戶 ssoSecret 簽 userToken 達成免登入。
  3. 在頁面貼上載入碼：
    <script src="https://你的網域/widget.js" data-key="pk_…"></script>

本頁即為示範：已自動帶入 SSO，點右下角小幫手就能直接對話。
