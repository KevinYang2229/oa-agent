不用碰後端，貼一段 script 網站就有 AI 小幫手

這版完全不做 SSO 免登入：頁面只貼一行 <script>，不需要任何後端、不需要 SDK、不需要簽 token。使用者點開小幫手後，在 widget 內自行登入即可。

試試畫面 右下角 的小幫手 ↘

⚡ 一行嵌入

<script src=".../widget.js"> 貼上即用，純 JS、零依賴、不必打包。

🚫 不用後端

不簽 token、不放 ssoSecret、不用動你的系統。只要能放一段 HTML 就行。

🔑 使用者自行登入

不沿用宿主身分；首次點開 widget 時由使用者在聊天室內登入。

🎨 可設定外觀

data-title / data-theme / data-locale / data-position 調整文案、亮暗、語言、左右。

🛡️ 樣式隔離

聊天介面跑在 iframe 內，scoped 樣式不污染宿主頁，不會被你的 CSS 影響。

🧩 事件可監聽

送出等事件以 oa-agent:submitted 派發，宿主頁可接手後續流程。

怎麼接？兩步

  1. 跟我們申請租戶，拿到公開金鑰 pk_…（pk 是公開的，可放前端）。
  2. 在頁面任意位置貼上這一行就好：
    <script src="https://你的網域/widget.js" data-key="pk_…" data-title="Acme 小幫手"></script>

沒有第三步：不需要後端端點、不需要 data-user-token。想要「沿用宿主登入身分、免再登入」的版本，見 ../widge-tenant（需要一個後端簽 token 的端點）。
